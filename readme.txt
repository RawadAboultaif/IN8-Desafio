Instruções para visualizar o site.

1- Baixar e instalar o nodejs https://nodejs.org
2- Abrir o cmd e localizar a pasta onde se encontram
os arquivos pelo comando 'cd' + diretorio onde estao localizados os arquivos.
3- Usar o comando 'npm instal' e aguardar a instalação.
4- Usar o comando 'npm start'.
5- Espero que goste !!
